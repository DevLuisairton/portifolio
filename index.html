Projetos em Destaque

Aqui estão alguns dos meus trabalhos

  • SertifySimulator

    O SertifySimulator é um projeto que desenvolvi para auxiliar na preparação para certificações, especialmente da AWS. Trata-se de um site de simulados e trilhas de estudo, oferecendo uma experiência completa para quem deseja testar seus conhecimentos antes da prova. A plataforma conta com funcionalidades como temporizador de exame, explicações detalhadas das respostas, cálculo de porcentagem de acertos e muito mais. Atualmente, estou expandindo o conteúdo, adicionando novos simulados para a certificação AWS Cloud Practitioner e, em breve, para outras certificações da AWS. O projeto continua em desenvolvimento, sempre evoluindo para proporcionar uma experiência de aprendizado cada vez melhor.

    Leia mais →

  • AutoNimbus

    Este projeto foi desenvolvido como parte do Trabalho de Conclusão de Curso (TCC) da Escola da Nuvem, onde aplicamos conceitos avançados de arquitetura em nuvem para criar uma solução completa de automação. O foco foi projetar uma infraestrutura escalável, segura e altamente disponível, utilizando serviços nativos da AWS para otimizar operações e reduzir custos.

    Leia mais →

  • Jogo do amigo secreto

    O jogo do Amigo Secreto é uma aplicação web simples e interativa, desenvolvida JavaScript. Ele permite que o usuário adicione nomes de participantes, visualize a lista atualizada e realize o sorteio de um amigo secreto aleatoriamente.

    Leia mais →

  • Jogo do número secreto

    O Jogo do Número Secreto é um jogo desenvolvido em JavaScript que desafia o usuário a adivinhar o número aleatório escolhido pela máquina. Durante o jogo, a aplicação fornece dicas de forma interativa, indicando se o número informado é maior ou menor que o número secreto. Além disso, o jogo conta com suporte de voz, tornando a experiência mais dinâmica e envolvente.
